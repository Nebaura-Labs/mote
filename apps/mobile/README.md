# Mote Mobile App

**Coming Soon**

Expo/React Native bridge app that connects Mote hardware to your clawd.bot instance.

## Features (Planned)

- Bluetooth/WiFi connection to Mote device
- clawd.bot API integration
- Device pairing and setup
- Voice command routing
- Device settings and configuration

## Development

```bash
# From root
pnpm install

# Start Expo
cd apps/mobile
pnpm start
```
